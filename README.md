# unx-utils
javascript基础工具集合
## 模块功划分
### array
|  工具函数名   | 功能  |
|  ----  | ----  |
| arrayEqual  | 判断两个数组是否相等 |
### cookie
|  工具函数名   | 功能  |
|  ----  | ----  |
| setCookie  | 设置cookie |
| getCookie  | 获取cookie |
| removeCookie  | 删除cookie |
### url
|  工具函数名   | 功能  |
|  ----  | ----  |
| urlQuery2Object  | 解析url的参数以对象的形式返回 |
| object2UrlQuery  | 将一个对象转成name=mike&age=6这种url参数字符串

## 关于测试
本项目的工具函数代码都会做单元测试，覆盖率争取达到95%以上。 |
